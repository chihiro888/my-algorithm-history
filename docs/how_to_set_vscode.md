# how to set VSCode

## install gcc

for windows
- https://www.mingw-w64.org/downloads/

check installed program
```bash
gcc -v
```

## install extensions
- C/C++
- C/C++ Extension Pack
- C/C++ Themes
- Code Runner
build and run
```
Ctrl + Alt + N  
```
